Using AI to Generate Music from Images

What would your favourite photo sound like?

ThemeSong answers a simple question: given an image — a pet, a memory, a place, a person — what music would it be? Upload a photo, and within a minute you get back a full song generated to match it.

Under the hood, OpenAI interprets the image and crafts a detailed prompt for Suno, which then generates the track. The whole thing runs on a Raspberry Pi in my living room, exposed via a Cloudflare tunnel, with a Python backend handling image ingestion, user accounts, and API orchestration.

Try it: themesong.florentin.ca — this is a rough demo running on hobby hardware. Feedback welcome!
